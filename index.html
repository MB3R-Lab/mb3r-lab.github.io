What we do

MB3R Lab is an R&D group working on automatic model discovery and simulation-driven resilience analysis for distributed systems (e.g., microservices).

  • Model Discovery — synthesize system dependency models from configuration, traces, and telemetry.
  • What-if Simulation — run Monte-Carlo failure scenarios on the discovered model without touching production.
  • Causal Resilience — sheaf-theoretic analysis and Effective Information (EI) to identify macro-structures that improve predictability and control.
